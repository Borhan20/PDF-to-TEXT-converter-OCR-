# OCR PDF Converter

A Python-based CLI tool to convert PDF files (scanned or searchable) into plain text using Tesseract OCR.

## Setup

1. Create a virtual environment:
   ```bash
   python3 -m venv .venv
   source .venv/bin/activate
   ```

2. Install dependencies:
   ```bash
   pip install .
   ```

3. (System Dependencies) Ensure Tesseract OCR and Poppler are installed on your system.
